# 💰 Pocket Planner - Financial Dashboard

A modern financial dashboard to track transactions, analyze spending, and gain insights.

## 🚀 Features
- Dashboard with balance, income, and expense summary
- Interactive charts for trends and category breakdown
- Transactions table with filtering and search
- Role-based UI (Admin / Viewer)
- Insights on spending patterns
- Persistent state using localStorage

## 🛠️ Tech Stack
- React + Vite
- TypeScript
- TailwindCSS
- Zustand
- Recharts

## 📦 Setup
```bash
npm install
npm run dev
